Web Font Delivery

There are various techniques for providing fonts to a user's web browser when developing a website. One strategy involves the use of web fonts, which enables the retrieval of fonts from a server when a webpage is accessed.

To use a web font, you have to change it into a format that works well on the web, like WOFF or WOFF2. After that, you can put these font files on a web server, just like you would with images or other stuff. In your website's CSS, you can then say which font to use using something called "@font-face," and this tells the browser where to get that font.

When a browser asks for a web page, it also grabs the web font files at the same time, along with all the other stuff needed for the site. This way, the browser has the right fonts to show text with the style you want, and you're not limited to the standard fonts on the user's system. This method offers a straightforward way to use your chosen fonts online. It can make the page a bit bigger to load, so it's good to use optimization tricks like font subsetting. But overall, web fonts give you a lot of control over how your text looks on the web, which is great for branding and design purposes
